Sobre mí

Soy Jorge Salas, un creador enfocado en desarrollar experiencias digitales, desde plataformas web modernas hasta videojuegos interactivos. Me gusta crear proyectos que no solo funcionen bien, sino que realmente se sientan fluidos, rápidos y profesionales.

Web Specialist

Desarrollo desde sitios personalizados con HTML, CSS y JavaScript hasta soluciones dinámicas en WordPress. Mi prioridad es que cada página sea totalmente responsiva, esté optimizada para buscadores (SEO) y ofrezca una navegación impecable.

Game Developer

Me enfoco en crear mecánicas sólidas y experiencias envolventes utilizando motores como Unity y Unreal Engine. Esta faceta me permite dominar la lógica de programación compleja; esa misma atención al detalle la traslado a mis desarrollos web.

Hardware & Logic

También disfruto trabajar con hardware y Arduino, integrando el mundo físico con el digital. Mi enfoque es simple: entregar proyectos bien hechos, con un diseño atractivo y un rendimiento que supere las expectativas.
